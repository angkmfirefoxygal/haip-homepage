Haip

URL 하나로 AI가 QA를 끝냅니다

개발자 없이도, 스크립트 없이도. AI가 실제 사용자처럼 사이트를 돌아다니며
버그를 찾고, 심각도를 분류하고, 재현 경로까지 정리해드립니다.

시작하기 GitHub ↗ NPM ↗
$ npx haip-agent

# Features

## 브라우저 자동화

AI 브라우저 조작
클릭, 입력, 스크롤, 페이지 이동을 자연어 지시로 수행합니다. Playwright 기반으로 실제 브라우저를 구동합니다.
실시간 스트리밍
테스트 진행 중 스크린샷과 동작 로그가 실시간으로 전달됩니다. 무슨 일이 일어나는지 바로 확인할 수 있습니다.
자연어 지시
"결제 흐름을 집중적으로 테스트해줘" 같은 자연어로 테스트 범위를 지정할 수 있습니다.

## QA 리포팅

심각도 자동 분류
발견된 버그를 HIGH / MEDIUM / LOW로 자동 분류합니다. 우선순위를 바로 파악할 수 있습니다.
재현 경로 기록
버그 발생까지의 사용자 여정을 단계별로 기록합니다. 개발팀이 바로 재현하고 수정할 수 있습니다.

# Getting Started

URL을 입력하거나 npx로 실행하세요. 어떤 웹 서비스에도 바로 적용됩니다.

웹에서 바로 사용

설치 없이 브라우저에서 URL 하나 입력하면 됩니다.

# 브라우저에서 바로 접속
$ open https://haip-flow-qa.base44.app
# URL 입력 → AI가 자동으로 테스트 수행
$ 결과는 1분 안에 확인 가능
CLI로 사용

개발 환경에서 npx 한 줄로 실행합니다.

# 설치 없이 바로 실행
$ npx haip-agent
# URL을 직접 지정해서 실행
$ npx haip-agent --url https://your-site.com
# 추가 지시사항과 함께 실행
$ npx haip-agent --url https://your-site.com --instruction "결제 흐름 테스트"
CI/CD 통합 (예정)

GitHub Actions와 연동해 배포 전 자동 QA를 파이프라인에 넣을 수 있습니다.

# .github/workflows/qa.yml
$ npx haip-agent --url $STAGING_URL --ci
